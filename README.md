# weiqi
A H5 weiqi(GO) game with network support

# install
```
git clone https://github.com/hijack-x/weiqi.git

cd weiqi
npm install

start.bat    # windows
start.sh     # linux

http://localhost:8080/          # 19*19
http://localhost:8080/?size=13  # 13*13
```

# 联网对弈说明
```
1、启动服务端(start.bat/start.sh)
2、把访问地址(http://<IP>:8080/)告诉对方
3、双方加入同一个房间
4、各自选择好棋子颜色，对弈自动开始
```
